Question-1: What are the different ways to select an element in the DOM?

getElementById(), getElementsByClass(), getElementsByTagName(), querySelector(), querySelectorAll(), document.body, document.documentElement

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML is used for get and edit or add html in an element, innerText is used for get and set only visible text inside the element, textContent is used for get and set all html including hidden element

Question-3: What is event delegation in the DOM?

Event delegation is a js technique which is use for adding single event listener to handle events in child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts on the element you interacted with and then moves up to its parent, grandparent, and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating, adding, and removing elements in JavaScript means making new elements, placing them inside the webpage, and deleting them when needed.
